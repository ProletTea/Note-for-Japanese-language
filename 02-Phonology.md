# 音系学
## 1.元音
现代日语的元音有五个：ア、イ、ウ、エ、オ。 
| 假名 | 国际音标 | 描述 |
| -------- | -------- | -------- |
| ア | [ä] | 开央不圆唇元音 |
| イ | [ɪ̟] | 闭前不圆唇元音 |
| ウ | [] | TODO |
| エ | [e̞] | 中前不圆唇元音 |
| オ | [o̞͑] | 半闭后圆唇元音 |
